
## Only

### Установка и запуск

```bash
git clone https://github.com/sanyOoOk/timeline-only-project.git

cd react-timeline-project
```

Установка зависимостей:
```bash
npm install
```

Запуск проекта в режиме разработки:
```bash
npm run start
```
После запуска проект будет доступен по адресу:
🔗 http://localhost:3000


Сборка:
```bash
npm run build:prod
```

Собранная версия появится в папке build/.
